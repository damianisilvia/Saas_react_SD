# 🚀 SaaS React SD

Benvenuto in **SaaS React SD**, un'applicazione web moderna, performante e reattiva sviluppata in React. Questo progetto rappresenta un esempio pratico di sviluppo software moderno, integrando metodologie di programmazione assistita da IA ed un'infrastruttura cloud di livello enterprise.

👉 **[Clicca qui per vedere l'applicazione online](https://saas-react-sd.damianisilvia1.workers.dev/login)**

---

## 🛠️ Architettura & Tech Stack

Il progetto è stato strutturato separando nettamente la logica di frontend, la persistenza dei dati e i processi di automazione:

* **Frontend Framework:** React (Functional Components ed React Hooks) con **Vite** come build tool per garantire un caricamento ultra-rapido (HMR) e performance ottimizzate in produzione.
* **Database & Autenticazione:** Integrazione in corso con **Supabase** (PostgreSQL) per la gestione sicura degli utenti, sessioni di login e persistenza dei dati in tempo reale.
* **Hosting & Cloud Infrastructure:** Distribuito globalmente tramite **Cloudflare Pages / Workers**, sfruttando una pipeline di *Continuous Integration & Continuous Deployment* (CI/CD) automatizzata direttamente dai push su GitHub.
* **AI-Assisted Development:** Sviluppato all'interno dell'IDE **Antigravity**, sfruttando le potenzialità del suo **Agente IA integrato** per l'ottimizzazione del codice, il refactoring del CSS responsive e il debugging rapido delle dipendenze.

---

## 🎨 Caratteristiche dell'Interfaccia (UI/UX)

* **Design Dark Moderno:** Interfaccia scura ad alto contrasto con accenti neon ciano e viola, ispirata ai moderni trend tecnologici.
* **Approccio Mobile-First:** Ottimizzazione nativa per smartphone, inclusa una *Bottom Navigation Bar* ergonomica e scritte ridimensionate dinamicamente per massimizzare la leggibilità.
* **Effetti Visivi Avanzati:** Layout basati su griglie bento, pannelli in stile *Glassmorphism* (sfocatura dello sfondo tramite proprietà CSS) e micro-animazioni fluide.
* 🤖 **Prototipo STITCH:** Collegamento diretto con **OpenAI Agent** e importazione dinamica del file `DESIGN.md` per l'allineamento del layout.
